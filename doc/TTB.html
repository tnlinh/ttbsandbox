The main class for consuming TTB web services.
Name Type Description
config Object The configuration info required
Name Type Default Description
partnerKey String The Partner-Key is a unique ID assigned to your account to be sent on every request.
baseURL String "https://direct.api.titletoolbox.com/" optional The Base URL to be used for APIs calls
debug String true optional SDK debug mode flag useful for logs, etc.
Returns:
Type Description
Object ttb - The instance associated with the provided configuration.
Examples
// With basic and minimum requirement.
 var ttb = new TTB({
   partnerKey: '{your partner key}',
 });

 
// With advanced configuration for custom URL, and logs for debug mode.
 var ttb = new TTB({
   partnerKey: '{your partner key}',
   baseURL: 'https://direct.api.titletoolbox.com/',
   debug: true
 });
